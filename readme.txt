== Description ==

This is a plugin that will display all Runescape Highscores for anyone with a Runescape username and is on the highscores list.

== Installation ==

Installing should be a piece of cake and take fewer than 2 minutes.

1. Login as admin to your wordpress site.
2. Upload the zip file from the top of this page.
3. Activate the plugin through the ‘Plugins’ menu in WordPress.
4. Go to the widgets menu (under the ‘Appearance’ menu in Wordpress).
5. Drag the RS Highscores widget to your sidebar and type your Runescape name in the control panel.
6. You are done! Go check out your page!
